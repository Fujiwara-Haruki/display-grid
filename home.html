Diplay: grid

O Grid Container é uma tag que envolve itens do grid, ao indicar display: grid, essa tag passa a ser um Grid Container.


  1. Display

    2. A função display: grid define um elemento como um Grid Container, exemplos:

    display: grid

  2. Grid Template Columns

    3. Define o número total de colunas que serão inseridas no grid, exemplos:



    grid-template-columns: 1fr 2fr: Desta forma Duas colunas serão criada, a segunda com o dobro de tamanho da primeira e o contéudo dentro do grid é respeitado, se o conteudo da primeira coluna for maior que o da segunda, a segunda coluna se torna maior.

  3. Grid Template Rows

    4. Usado para definir a quantidade de linhas no grid.


    grid-template-rows: 50px 100px 50px 150px: Vai criar um grid, sendo a primeira e a terceira com 50px, a segunda com 100px e a última com 150px.

  5. Grid Template

    6. Usamos o grid-template como atalho para definir grid-template, grid-template-rows e grid-template-areas, por exemplo:

    grid-template: 50px 1fr / 150px 1fr 150px: linhas: altura-50px, largura-1fr / colunas largura: 1° coluna, 2° coluna, 3° coluna.
